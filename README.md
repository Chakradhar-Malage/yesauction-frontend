# YesAuction - Online Auction Platform

YesAuction is a modern full-stack auction platform where users can create, browse, and participate in auctions in real time.

## Features

* User authentication with login and registration
* Create, edit, and manage auctions
* Browse auctions by category
* Real-time bidding using WebSocket
* User profile with edit functionality
* Seller dashboard
* Live auction countdown
* Responsive user interface

## Tech Stack

* Frontend: React, TypeScript, Tailwind CSS
* Routing: React Router
* API Handling: Axios
* State Management: React Hooks
* Real-time Communication: WebSocket

## Project Structure

* Components: Reusable UI components
* pages: Application pages
* api: API configuration and calls
* hooks: Custom React hooks
* types: TypeScript interfaces
* utils: Utility functions

## Getting Started

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm start
```
